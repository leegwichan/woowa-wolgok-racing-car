## 구현할 기능 목록

- InputView
  - [ ] 공통 : 입력 요청 메세지를 출력함
  - [ ] 공통 : 입력 형식이 일치하지 않은 경우, `IllegalArgumentException`을 발생시키고 다시 값을 입력 받음
  - [ ] 자동차 이름을 입력 받음
    - 입력 형식 : 자동차 이름은 쉼표(,)를 기준으로 구분하며 이름은 5자 이하
  - [ ] 시도 횟수를 입력 받음
    - 입렫 형식 : 0 보다 큰 숫자


- OutputView
  - [ ] 정보를 받아 현재 차량 진행 상황을 이름과 함께 출력함
    ```text
    pobi : --
    woni : ----
    jun : ---
    ```
  - [ ] 정보를 받아 우승자 안내 문구를 출력
    ```text
    최종 우승자 : pobi, jun
    ```

- Car
  - [ ] 한 칸 움직이는 기능
  - [ ] 차의 이름을 전달함
  - [ ] 차의 현재 위치를 전달함

- CarGame
  - [ ] 각각 0~9 사이의 랜덤의 숫자를 받아 4 이상일 경우 차를 이동함
  - [ ] 현재 차들의 위치 상태를 제공함
