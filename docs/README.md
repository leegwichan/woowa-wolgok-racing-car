## 📋 기능목록

- [ ] 자동차 이름을 입력받는다.
  - 입력값을 검증한다.
  - [ ] 이름은 1자 이상 5자 이하이다.
  - [ ] 이름의 갯수는 int 범위를 벗어나지 않는다.


- [ ] 이동 횟수를 입력받는다.
  - 입력값을 검증한다. 
  - [ ] 이동 횟수는 음수가 아니다.
  - [ ] 이동 횟수는 int 범위를 벗어나지 않는다.


- [ ] 0-9 사이 무작위 값을 생성한다.


- [ ] 자동차 별 무작위 값을 구해 4이상이면 전진한다.
  - 이동 횟수만큼 반복한다.


- [ ] 이동 횟수마다 전진 결과를 출력한다.

- [ ] 가장 많이 전진한 자동차들(최종 우승자)을 찾는다.

- [ ] 최종 우승자를 출력한다.
